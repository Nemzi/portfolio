Technologies and future plans

Technologies used to create this portfolio are displayed below. You can choose to interact with one or several of the element display boxes, the ever changing colors are meant to represent that none of these technologies can make up a website on it's own.

HTML
The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser.
CSS3
Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML.
BS-5
Bootstrap is a free and open-source collection of CSS and JavaScript/jQuery code used for creating dynamic layout websites and web applications.
JS
JavaScript is a dynamic computer programming language. It is lightweight and most commonly used as a part of web pages.
